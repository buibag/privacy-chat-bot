2. What data we collect

  • Messages & interactions: text, quick replies, postback payloads, timestamps, delivery/read events, conversation IDs.
  • Attachments: images, video, audio, files you send.
  • PSID: the Page-Scoped ID that lets the bot reply.
  • Basic profile (if retrieved via User Profile API): first_name, last_name, profile_pic, locale, timezone.
  • Flow inputs: Orders (category, product, quantity, name, phone, address); Bookings (service, date, time, name, phone).
  • Technical logs: request/response payloads and status codes for troubleshooting/security.

We do not intentionally collect sensitive data.

3. Where data is stored
